# Task Management App

A  Task Management application where users can **register**, **login**, and **manage their tasks**.  
Only users with the **correct username and password** are allowed to log in.

---

##  Features

-  User Registration
-  User Login with Password Validation
-  Add Tasks (Title & Description)
-  Delete Tasks Anytime
-  Tasks are visible only to the logged-in user


---

## Tech Stack

- **Frontend:** React
- **Language:** TypeScript
- **Build Tool:** Vite
- **Package Manager:** npm

---

## Task Details

Each task contains:
- **Title**
- **Description**
